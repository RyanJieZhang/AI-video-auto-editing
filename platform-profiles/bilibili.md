# Bilibili Platform Profile

## Format

- Common ratios: 16:9 for standard uploads, 9:16 for vertical feed.
- Good explainers can run longer than short-form platforms, but the first 5 seconds still matter.

## Hook

- Start with a clear problem or strong claim.
- Avoid over-polished ad language if the target audience is technical or student-heavy.
- Show the result or key conflict early.

## Captions

- Captions should be readable on desktop and mobile.
- Use concise sentence chunks, not dense paragraph subtitles.
- Technical terms can stay on screen slightly longer.

## Pacing

- Allow more explanation than Douyin/Xiaohongshu.
- Use chapter markers, data cards, screen recordings, and comparison shots.

## Packaging

- Title can be explicit and search-friendly.
- Description should include workflow, tools, and project links.
- Cover should communicate the core conflict in one line.
