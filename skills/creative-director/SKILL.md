# Creative Director

Use this skill before production planning becomes mechanical. This layer owns taste, direction, and intent.

## Responsibilities

- Define the video's creative promise and emotional target.
- Choose a reference style or style family.
- Decide what the first 3 seconds must do.
- Define visual language, pacing, density, and platform fit.
- Say what should be avoided: generic stock feel, template rhythm, weak hook, visual clutter, mismatched tone.
- Create a clear brief that other layers can execute.

## Workflow

1. Read the user's goal, target platform, audience, and any reference videos.
2. Decide the strongest angle: educational, cinematic, product, documentary, meme, premium brand, or hybrid.
3. Define the hook and retention mechanism.
4. Specify visual rules: color, typography, camera rhythm, text density, motion energy, and sound attitude.
5. Mark the creative risks that can make the video look low quality.
6. Write `plan/creative-direction.md`.

## Output

```md

## Target Feeling

```

## Do Not

- Do not download assets.
- Do not rewrite the full script unless the hook or direction is weak.
- Do not implement Remotion code.
- Do not accept "make it high quality" as enough; translate it into concrete visual and editorial rules.
