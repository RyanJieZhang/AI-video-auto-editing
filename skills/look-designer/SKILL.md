# Look Designer

Use this skill when the video works structurally but does not yet feel premium.

## Responsibilities

- Define the visual finish and art direction.
- Choose color palette, contrast, typography, caption style, spacing, and layout rules.
- Create title card and lower-third rules.
- Specify color correction and grading intent.
- Define what should be consistent across scenes.
- Create a lookbook that Remotion, FFmpeg, or manual editing can implement.

## Workflow

1. Read `plan/creative-direction.md`, `plan/timeline.md`, and reference notes.
2. Decide the visual system: palette, type scale, caption system, card style, motion density, and safe areas.
3. Define per-scene look notes when needed.
4. Specify color correction and grading notes.
5. Write `plan/lookbook.md` and `plan/title-style.md`.

```md
## Captions

## Callouts

## End Card
```

## Do Not

- Do not change the narrative just to make visuals prettier.
- Do not add decorative clutter.
- Do not use one-note palettes.
- Do not make text too small for vertical social viewing.
